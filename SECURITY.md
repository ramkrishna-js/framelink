# Security Policy

## Supported Versions

The following versions of FrameLink are currently being supported with security updates:

| ------- | ------------------ |

## Reporting a Vulnerability

If you discover a security vulnerability within FrameLink, please send an e-mail to Ramkrishna via ramkrishnagithub@gmail.com. All security vulnerabilities will be promptly addressed.
